2.1 Linux Kernel

The foundation of Android, providing:

  • Drivers for camera, display, sensors, Bluetooth
  • Memory & process management
  • SELinux-based security

2.2 HAL ‚Äì Hardware Abstraction Layer

The HAL provides standard interfaces for hardware components (camera, audio, sensors, Bluetooth). This lets higher-level APIs interact with hardware without knowing how it physically works.

2.3 Native Libraries

High‚Äëperformance components written in C/C++:

  • OpenGL ES (graphics)
  • libmedia (audio/video)
  • SQLite (database)
  • SSL (security)

2.4 Android Runtime (ART)

ART replaced Dalvik and provides:

  • AOT + JIT compilation
  • Fast garbage collection
  • Improved debugging & crash reporting
  • Optimized DEX execution

2.5 Application Framework

The layer developers use daily. It includes:

  • Activity Manager
  • Notification Manager
  • Content Providers
  • Window Manager
